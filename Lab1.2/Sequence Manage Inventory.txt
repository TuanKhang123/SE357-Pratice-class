@startuml
title Quản Lý Kho Hàng Sequence Diagram

actor Admin as A
boundary "Inventory Management UI" as IMUI
control "Inventory Management Controller" as IMC
entity "Inventory Database" as ID

A -> IMUI: Truy cập giao diện quản lý kho hàng
IMUI -> IMC: Yêu cầu quản lý kho hàng
IMC -> ID: Truy vấn thông tin kho hàng
ID --> IMC: Thông tin kho hàng
IMC --> IMUI: Hiển thị thông tin kho hàng

A -> IMUI: Xem số lượng hàng tồn kho
IMUI -> IMC: Yêu cầu xem số lượng
IMC -> ID: Xem số lượng hàng tồn kho
ID --> IMC: Số lượng hàng tồn kho
IMC --> IMUI: Hiển thị số lượng hàng tồn kho

A -> IMUI: Xem chi phí nhập kho
IMUI -> IMC: Yêu cầu xem chi phí nhập
IMC -> ID: Xem chi phí nhập kho
ID --> IMC: Chi phí nhập kho
IMC --> IMUI: Hiển thị chi phí nhập kho

A -> IMUI: Xem trạng thái trong kho hàng
IMUI -> IMC: Yêu cầu xem trạng thái
IMC -> ID: Xem trạng thái trong kho hàng
ID --> IMC: Trạng thái trong kho hàng
IMC --> IMUI: Hiển thị trạng thái trong kho hàng
@enduml
