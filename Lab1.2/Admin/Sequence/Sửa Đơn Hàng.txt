@startuml
actor Admin
participant "OrderManagementSystem" as OMS
database Database

Admin -> OMS: Yêu cầu xem đơn hàng
OMS -> Database: Truy vấn thông tin đơn hàng
Database --> OMS: Trả về thông tin đơn hàng
OMS --> Admin: Hiển thị thông tin đơn hàng

Admin -> OMS: Yêu cầu sửa đơn hàng (ID, thông tin mới)
OMS -> Database: Kiểm tra đơn hàng có thể sửa không
alt đơn hàng có thể sửa
    Database -> OMS: Xác nhận có thể sửa
    OMS -> Database: Cập nhật thông tin đơn hàng
    Database --> OMS: Xác nhận cập nhật
    OMS --> Admin: Hiển thị thông báo thành công
else đơn hàng không thể sửa
    Database -> OMS: Thông báo không thể sửa
    OMS --> Admin: Hiển thị thông báo không thể sửa đơn hàng
end

@enduml
